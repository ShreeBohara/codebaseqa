# Support

## Where to Get Help

- Usage questions: open a GitHub issue with context and steps tried.
- Bug reports: use the Bug Report issue template.
- Feature requests: use the Feature Request issue template.
- Security concerns: follow `SECURITY.md` and avoid public disclosure.

## What to Include

- What you are trying to do
- Expected behavior
- Actual behavior
- Reproduction steps
- Logs/screenshots (if relevant)
- Environment details (OS, Node/Python versions, provider setup)

## Response Model

This is an open-source project maintained with best effort.
Response times may vary, but clear and reproducible reports are prioritized.
